## Guia de APIs

Use esta documentação para integrar:

- Geração de cobrança (CASHIN)
- Consulta de transações
- Transferências/saques (CASHOUT)
- Consulta de saldo
- Webhooks de atualização

## Fluxo recomendado

<AccordionGroup>
  <Accordion title="1. Autenticação" icon="key">
    Gere sua <code>sk_live_prod_*</code> no painel e use no header <code>Authorization: Bearer ...</code>.
  </Accordion>
  <Accordion title="2. Gerar transação PIX" icon="qrcode">
    Chame <code>POST /v1/integrations/create-transaction</code> e exiba o <code>pixCode</code> para pagamento.
  </Accordion>
  <Accordion title="3. Acompanhar status" icon="list-check">
    Consulte por <code>transactionId</code> no endpoint <code>GET /v1/integrations/list-transactions</code>.
  </Accordion>
  <Accordion title="4. Saques e comprovante" icon="arrow-right-arrow-left">
    Para saque, use <code>POST /v1/integrations/create-cashout</code>. Quando concluir, use o <code>receiptUrl</code> HTML.
  </Accordion>
</AccordionGroup>
