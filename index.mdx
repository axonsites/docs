## Fluxos principais

<AccordionGroup>
  <Accordion title="CASHIN (cobranca)" icon="money-bills">
    1. Chame <code>POST /v1/integrations/create-transaction</code>.
    2. Salve o <code>transactionId</code> retornado.
    3. Consulte status em <code>GET /v1/integrations/list-transactions?transactionId=...</code>.
    4. Aguarde <code>COMPLETED</code> para considerar pagamento confirmado.
  </Accordion>
  <Accordion title="CASHOUT (saque)" icon="arrow-right-arrow-left">
    1. Chame <code>POST /v1/integrations/create-cashout</code>.
    2. O saque e enviado para a Woovi com aprovacao imediata.
    3. Consulte em <code>GET /v1/integrations/cashout/:id</code>.
    4. Quando concluido, <code>receiptUrl</code> aponta para comprovante HTML.
  </Accordion>
  <Accordion title="Conciliação" icon="list-check">
    Use sempre <code>transactionId</code> (UUID) no seu sistema. O ID inteiro e legado.
  </Accordion>
</AccordionGroup>
