---
title: "Gerar cobranca"
openapi: "POST /v1/integrations/create-transaction"
---

Cria uma cobranca de entrada e retorna o `transactionId` publico.

<Info>
  Para PIX, a resposta inclui `pixCode`.
  Para cartao via Stripe, a resposta inclui `stripeCheckout`.
  Se quiser atribuicao de campanha, envie `tracking_parameters` com seus UTMs.
</Info>

<Info>
  Se voce estiver implementando cartao Stripe, veja tambem a pagina [Cartao (Stripe)](/stripe-card).
</Info>
