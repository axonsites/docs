## Base URL

`https://api.axnpay.com.br/v1`

## Autenticação da Integrations API

Todas as rotas em `/v1/integrations/*` usam:

```http
Authorization: Bearer sk_live_prod_xxxxxxxxxxxxxxxxx
```

<Warning>
  Nunca exponha sua `sk_live_prod_*` no frontend.
</Warning>

## IDs de transacao

- `transactionId` (UUID): identificador público recomendado para conciliação.
- `id` inteiro: legado, mantido para compatibilidade.

## Escopo desta referencia

Esta seção cobre endpoints para:

- Criação de cobrança (`CASHIN`)
- Consulta de transações
- Criação e consulta de saque (`CASHOUT`)
- Consulta de saldo
- Fluxo de aprovação legado de cashout
