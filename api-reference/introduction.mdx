## Base URL

`https://api.axnpay.com.br/v1`

## Autenticacao da Integrations API

Todas as rotas em `/v1/integrations/*` usam:

```http
Authorization: Bearer sk_live_prod_xxxxxxxxxxxxxxxxx
```

<Warning>
  Nunca exponha sua `sk_live_prod_*` no frontend.
</Warning>

## IDs de transacao

- `transactionId` (UUID): identificador publico recomendado para conciliacao.
- `id` inteiro: legado, mantido para compatibilidade.

## Escopo desta referencia

Esta secao cobre endpoints para:

- Criacao de cobranca (`CASHIN`)
- Consulta de transacoes
- Criacao e consulta de saque (`CASHOUT`)
- Consulta de saldo
- Fluxo de aprovacao legado de cashout
